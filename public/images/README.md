# Areli Jewellery — Image Assets

Place your local images in the folders below. The site will automatically use them once filenames match.

## Folder Structure

| Folder | Purpose | Suggested filenames |
|--------|---------|---------------------|
| `hero/` | Hero banner — model wearing statement piece | `hero-main.jpg` |
| `products/` | Product photos for carousel & bestsellers | `product-1.jpg` … `product-8.jpg` |
| `lifestyle/` | Style inspiration grid | `work.jpg`, `gym.jpg`, `night-out.jpg` |
| `about/` | About page imagery | `about-story.jpg` |

## Replacing placeholders

1. Copy your images into the matching folder above.
2. Use the suggested filenames, **or** update the `src` paths in `index.html`, `about.html`, and `js/products.js`.
3. Recommended size: **1200×1500px** for products, **1920×1080px** for hero/lifestyle.

The site currently uses high-quality placeholder images until your files are added.
